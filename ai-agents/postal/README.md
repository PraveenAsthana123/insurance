# Postal · Email Server

OSS mail server · self-hosted SendGrid-alternative · DKIM · bounce handling

License: MIT · Port: 5000 · Env: admin email

## Install (universal · preferred)

```bash
./scripts/setup_ai_agent_stack.sh --tool postal
```

Or per-tool:

```bash
./ai-agents/postal/deep/scripts/install.sh
```

Or manually: `docker compose up postalserver/postal`

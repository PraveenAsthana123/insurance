# OhMyForm · Survey/Forms

Self-hosted form builder · OSS alternative to Typeform · MongoDB-backed

License: AGPL-3 · Port: 5000 · Env: MongoDB

## Install (universal · preferred)

```bash
./scripts/setup_ai_agent_stack.sh --tool ohmyform
```

Or per-tool:

```bash
./ai-agents/ohmyform/deep/scripts/install.sh
```

Or manually: `docker run -p 5000:5000 ohmyform/ohmyform`

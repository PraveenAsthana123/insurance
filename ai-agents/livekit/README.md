# LiveKit · Voice Agent Platform

Real-time audio/video infrastructure for AI agents · WebRTC SFU + LLM agents SDK

License: Apache-2.0 · Port: 7880 · API keys: LIVEKIT_API_KEY · LIVEKIT_API_SECRET

## Install (universal · preferred)

```bash
./scripts/setup_ai_agent_stack.sh --tool livekit
```

Or per-tool:

```bash
./ai-agents/livekit/deep/scripts/install.sh
```

Or manually: `pip install livekit livekit-agents livekit-plugins-openai`

# Pipecat · Voice Agent Platform

Open-source Python framework for building voice/multimodal AI bots · Daily.co · pipeline-style

License: BSD-2 · Port: none · API keys: OPENAI_API_KEY · DAILY_TOKEN

## Install (universal · preferred)

```bash
./scripts/setup_ai_agent_stack.sh --tool pipecat
```

Or per-tool:

```bash
./ai-agents/pipecat/deep/scripts/install.sh
```

Or manually: `pip install pipecat-ai[daily,openai,silero]`

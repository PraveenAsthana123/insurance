# Keila · Email/Newsletter

OSS Elixir-based newsletter platform · GDPR-friendly · self-hosted Mailchimp-alternative

License: MIT · Port: 4000 · Env: SMTP creds

## Install (universal · preferred)

```bash
./scripts/setup_ai_agent_stack.sh --tool keila
```

Or per-tool:

```bash
./ai-agents/keila/deep/scripts/install.sh
```

Or manually: `docker run -p 4000:4000 ghcr.io/keila/keila`

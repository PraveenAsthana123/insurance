# Mailtrain · Email/Newsletter

OSS Node.js newsletter platform · campaign + list management · Amazon SES integration

License: GPL-3 · Port: 3000 · Env: SES creds

## Install (universal · preferred)

```bash
./scripts/setup_ai_agent_stack.sh --tool mailtrain
```

Or per-tool:

```bash
./ai-agents/mailtrain/deep/scripts/install.sh
```

Or manually: `docker run -p 3000:3000 mailtrain/mailtrain`
